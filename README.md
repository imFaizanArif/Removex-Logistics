# Removex-Logistics

Next.js app with **TypeScript**, **Tailwind CSS**, and **Framer Motion**.

## Stack

- [Next.js 14](https://nextjs.org/) (App Router)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` — start dev server
- `npm run build` — build for production
- `npm run start` — start production server
- `npm run lint` — run ESLint

## Project structure

- `src/app/` — App Router pages and layout
- `src/app/globals.css` — global Tailwind styles
- `src/app/page.tsx` — home page (includes Framer Motion example)

Use `"use client"` in any component that uses Framer Motion so it runs on the client.

## Deploy on Vercel

The project includes a `vercel.json` configuration. Deploy by pushing to GitHub and importing the repo in [Vercel](https://vercel.com), or run:

```bash
npx vercel
```
